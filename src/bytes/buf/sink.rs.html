Search Tricks


                

                    Prefix searches with a type followed by a colon (e.g.
                    fn:) to restrict the search to a given type.
                


                

                    Accepted types are: fn, mod,
                    struct, enum,
                    trait, type, macro,
                    and const.
                


                

                    Search functions by type signature (e.g.
                    vec -> usize or * -> vec)
