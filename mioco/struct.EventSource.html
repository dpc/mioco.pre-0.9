Event source inside a coroutine

Event sources are a core of Mioco. Mioco coroutines use them to handle IO in a blocking fashion.

They come in different flavours and can be created from native mio types by wrapping withing a coroutine with MiocoHandle::wrap() or type-specific constructors like mailbox() or MiocoHandle::timeout().
